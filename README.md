# E-commerce__Website
Création d'un Site e-commerce – Projet HTML/CSS

# Objectif
Créer une maquette fonctionnelle d’un site e-commerce en utilisant uniquement HTML et CSS.
L’objectif était de travailler la structure des pages, la mise en forme et le design responsive.

# Technologies utilisées
	•	HTML5
	•	CSS3

# Fonctionnalités présentes
	•	Navigation entre les pages (index.html, produits.html, contact.html,etc...)
	•	Mise en page structurée (header, footer, sections produits)
	•	Styles CSS appliqués (couleurs, polices, mise en page responsive)

# Ce que j’ai appris
	•	Structurer un site web avec plusieurs pages
	•	Utiliser correctement HTML et CSS
	•	Organiser les fichiers du projet (index.html, css/, images/)
	•	Créer un design cohérent et responsive
  • Familiariser avec les outils de développement modernes, notamment VS Code pour l’édition du code et Live Server pour visualiser le rendu en temps réel.
